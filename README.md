## 프로젝트 후기

지금까지 하면서 너무 아쉬운 프로젝트였다 ㅠㅠ 강의도 생각보다 어렵고 이해하기가 쉽지 않아 시간을 많이 사용하였고
처음 배운 걸 바로 사용하려고 하니 너무 쉽지가 않았다. 아직 내가 많이 부족한 걸 느끼는 프로젝트였고 이 프로젝트를
기반으로 더욱더 성장해야겠다고 생각했다.
